14def equate_flows(model, flows1, flows2, factor1=1, name="equate_flows"): 

15 r""" 

16 Adds a constraint to the given model that sets the sum of two groups of 

17 flows equal or proportional by a factor for each timestep. 

18 

19 **The following constraints are built:** 

20 

21 .. math:: 

22 \text{factor\_1} \cdot 

23 \sum_{\text{flow in flows1}} \text{flow}_{t} = 

24 \sum_{\text{flow in flows2}} \text{flow}_t \forall t 

25 

26 Parameters 

27 ---------- 

28 model : oemof.solph.Model 

29 Model to which the constraint is added. 

30 flows1 : iterable 

31 First group of flows, to be set to equal with Var2 and multiplied with 

32 factor1. 

33 flows2 : iterable 

34 Second group of flows, to be set equal to (Var1 * factor1). 

35 factor1 : numeric, default=1 

36 Factor to define the proportion between the two groups. 

37 name : str, default='equate_flows' 

38 Name for the equation e.g. in the LP file. 

39 

40 Returns 

41 ------- 

42 the updated model. 

43 """

47 sum1_t = sum(m.flow[fi, fo, ts] for fi, fo in flows1) 

48 sum2_t = sum(m.flow[fi, fo, ts] for fi, fo in flows2) 

49 expr = sum1_t * factor1 == sum2_t

67def equate_flows_by_keyword( 

68 model, keyword1, keyword2, factor1=1, name="equate_flows" 

69): 

70 r""" 

71 This wrapper for equate_flows allows to equate groups of flows by using a 

72 keyword instead of a list of flows. 

73 

74 Parameters 

75 ---------- 

76 model: oemof.solph.Model 

77 Model to which constraints are added. 

78 keyword1: string 

79 keyword to consider (searches all Flows). 

80 keyword2: string 

81 keyword to consider (searches all Flows). 

82 factor1 : numeric, default=1 

83 Factor to define the proportion between the two groups. 

84 name : str, default='equate_flows' 

85 Name for the equation e.g. in the LP file. 

86 

87 Returns 

88 ------- 

89 the updated model 

90 

91 See Also 

92 -------- 

93 equate_flows(model, flows1, flows2, 

94 factor1=1, name="equate_flows") 

95 """ 

96 flows = {} 

97 for n, keyword in enumerate([keyword1, keyword2]): 

98 flows[n] = [] 

99 for i, o in model.flows: 

100 if keyword in model.flows[i, o].custom_properties: 

101 flows[n].append((i, o)) 

102 

103 return equate_flows(model, flows[0], flows[1], factor1=factor1, name=name)
