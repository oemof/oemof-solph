v0.6.2
------

API changes
###########

* The `Flow` parameter `bidirectional` now raises a `FutureWarning`,
  as it is to be removed in a following version.
* The `Flow` parameters `min` and `max` have been renamed to `minimum` and
  `maximum`, respectively. The old parameters are still available but raise a
  `FutureWarning`. Please use the new parameter names in future code.

New features
############

* Classes `Flow`, `NonConvex`, and `Investment` now use `custom_properties`
  in the way that `Node` classes (e.g. `Bus`, or `Converter`) do.

Documentation
#############

* The `Flow` parameters `min` and `max` are now listed with their default 
  values of 0 and 1, respectively, in the `Flow` API documentation.
* A tutorial focusing on time indexes and aggregation is added.

Bug fixes
#########

* Fixed that processing.results failed when using Pandas 3 with TSAM.

Other changes
#############

* Support Python 3.14 instead of Python 3.10.
