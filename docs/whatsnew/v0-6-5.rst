v0.6.5
------

API changes
###########

* The option `solph.Investment` now has the multiplication operator (`*`)
  defined. Immagine, you have a PV capacity of 500 W/panel, now you can give
  the investment costs per panel without having to calculate the specific
  costs: `nominal_capacity=Investment(ep_costs=200)  # 200 €/kW` and
  `nominal_capacity=0.5*Investment(ep_costs=100)  # 0.5 kW/panel, 100 €/panel`.
